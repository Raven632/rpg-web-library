# 🐉 RPG Web Library & Player

Self-hosted библиотека и веб-эмулятор для игр, созданных на движке RPG Maker (MV/MZ). Позволяет запускать ПК-версии игр прямо в браузере (включая мобильные устройства) с поддержкой виртуального геймпада и облачных сохранений.

## ✨ Главные фичи

* **Авто-Парсинг:** Закиньте ZIP-архив с игрой, и сервер сам найдёт обложку, оригинальное название и метаданные.
* **Математический CDN-хак:** Обходит блокировки провайдеров и защиту Cloudflare, вычисляя прямые ссылки на обложки (для игр с DLsite).
* **Мобильный Геймпад:** Встроенный D-Pad и кнопки действий (Shift, Enter, Esc, Menu) для игры со смартфона.
* **Облачные сохранения:** Ваши сейвы больше не привязаны к кэшу браузера. Они автоматически синхронизируются с сервером.
* **Smart Auto-Locator:** Сервер сам находит исполняемые файлы игры, даже если они спрятаны глубоко в папке `www`.
* **Устранение рассинхрона:** Фикс бага с мышью при нестандартном разрешении экрана.

## 🚀 Установка (Через Docker)

Для запуска вам понадобится только Docker и Docker Compose.

1. Склонируйте репозиторий:
   ```bash
   git clone [https://github.com/Raven632/rpg-web-library.git](https://github.com/ВАШ_НИК/rpg-web-library.git)
   cd rpg-web-library
2. Запустите контейнер: docker compose up --build -d
3. Откройте в браузере: http://localhost:8080

Как добавлять игры?

Просто нажмите кнопку «Добавить игру» в веб-интерфейсе и выберите ZIP/7z архив. Сервер сам распакует игру, применит патчи и выведет её в библиотеку.
